Arreglos y Matrices

En Java, los arreglos (Arrays) son estructuras de datos que almacenan múltiples valores del mismo tipo en una sola variable. Cada elemento en un arreglo tiene una posición o índice, que comienza en 0.

Declaración y uso de arreglos

Para declarar un arreglo se especifica el tipo de elementos que este va a almacenar: int, string, etc. Tambien se debe especificar la cantidad maxima que debe almacenar.

Si queremos agregar un elemento al arreglo debemos ubicarnos en una posición del mismo y asignarle un valor adecuado segun el tipo de arreglo que manejamos, para ello utlizamos las llaves [] y dentro el valor del indice o posición del arreglo al que queremos asignarle un valor. 


    int[] numeros = new int[5];
    numeros[0] = 10; 
    numeros[1] = 20;

También se pueden declarar e inicializar al mismo tiempo, con llaves y los valores separados por comas:


    String[] nombres = {"Ana","Luis","Carlos"}

Existe algunos metodos que se pueden aplicar directamente sobre el arreglo. Para implementarlos remplazar la palabra array, array1, array2 por los nombres de los arreglos que manejes

  • array.length: Propiedad que devuelve la longitud (número de elementos) del arreglo.
  • Arrays.sort(array): Ordena los elementos del arreglo de menor a mayor.
  • Arrays.toString(array): Devuelve una representación en forma de cadena del arreglo.
  • Arrays.copyOf(array, #Nuevo tamaño): Crea una copia del arreglo con una nueva longitud.
  • Arrays.equals(array1, array2): Compara si dos arreglos son iguales elemento por elemento.
  • Arrays.fill(array, #valor): Asigna un valor a todos los elementos del arreglo.

Recorrido de un arreglo

Para recorrer un arreglo debemos llamar cada valor del mismo con las llaves y el indice. La forma más comoda de recorrer el arreglo es con un bucle for: 


    int[] numeros = new int[5];
    for (int i = 0; i < numeros.length; i++) {
        System.out.println(numeros[i]);
    }

Arreglos multidimensionales (Matrices)

Las matrices son arreglos de dos o más dimensiones. En Java, una matriz de dos dimensiones puede verse como una tabla. Para definir las matrices, es igual a la Sintaxis de los arreglos pero con 2 o mas campos de valores. 


    int[][] matriz = new int[2][3];
    numeros[0][0] = 1; 
    numeros[0][1] = 2;
    numeros[1][0] = 3;

Las matrices se caracterizan por tener 2 dimensiones (i, j), se pueden comparar a una tabla con filas y columnas. Para poder acceder a un elemento tenemos que referirnos a la posición i,j para obtenerlo.

Recorrer una matriz:


    for (int i = 0; i < matriz.length; i++) {
        for (int j = 0; j < matriz[i].length; j++) {
            System.out.println("Elemento en ["+ i +"]["+ j +"]: "+ matriz[i][j];
        }
    }

Un ejemplo para el uso de matrices es cuando queremos guardar varios datos de un mismo objeto, en este caso personas. Definir una matriz donde las filas sea una persona distinta y las columnas sea la información personal de esa persona. 


    String[][] personas = {
        {"Chinatsu","18","Tokyo"},
        {"Hina","17","Kyoto"},
        {"Daniel","19","Tuluá :("}
    };
