Ciclos

Dentro de los fundamentos de programacion, una de las herrmaientas mas importantes y que nos acompañaran por mucho tiempo, son los ciclos. Los ciclos son estructuras que nos permiten repetir un mismo fragmento de codigo varias veces de manera automatica, podemos establacer la cantidad de veces que se ejecuta con una condicion o una variable.
Imagina que en un programa requerimos llevar a cabo una operación sencilla como sumar, pero tenemos que hacerla varias veces. Para ello utilizaremos for y while.

Ciclos For

Este bucle for es uno de los elementos que más se utiliza en programación. Con este tipo de ciclo puedes establecer el número de iteraciones que necesitas que tu bucle repita hasta que quieras que se acabe (debes conocer previamente el número de iteraciones).
Tiene 4 parámetros:

  • Variable de inicio: donde se comienza a ejecutar el ciclo (el nombre estándar de esta variable es i)
  • Condición o límite del ciclo: solo se repite si la condición es verdadera
  • Modificador: modificará la variable cuando se requiera
  • Cuerpo: donde se añaden las instrucciones a repetir


    for (int i = 0; i < n; i++) {
        // Instrucciones para ejecutar en el bucle
    }

En los bucles for se instancia una variable, que es conocida como iterador (i) o variable de inicio. Esta variable i es importante ya que define la cantidad de veces que se ejecuta el ciclo. Para el caso de este ejemplo se tomo int i = 0, estableciendo el valor inicial de i en 0. Ya que es una variable de iteración, cuando se realiza el bucle correctamente y se va a repetir, este iterador cambia segun le especifiquemos, en este caso: i++ se encarga de aumentar 1 al valor de i despues de cada ciclo. Detener un ciclo For

Para que un bucle for deje de ejecutarse se tiene un argumento en el cuerpo conocido como condicion de parada. Esta condicion es un booleano que permite que el codigo se repita siempre y cuando dicha condicion sea verdadera (True)

Para este ejemplo, la condicion de parada es i < n, una operación de comparación, donde el valor de n determinaria la cantidad de veces que veces que se repite el codigo en el ciclo. Cada vez que el ciclo se ejecuta, se evalua esta condición, ya que el valor de i cambia con cada iteración.
El códdigo que se ejecuta dentro del bucle tiene ciertas características:

  • - Las variables declaradas dentro, no se ven afectadas por el bucle a diferencia de las que se declararon fuera.
  • - Se pueden usar if para controlar condiciones específicas dentro del bucle.
  • - Es posible usar break para salir anticipadamente del bucle si se cumple una condición.
  • - Se puede usar continue para saltar una iteración sin salir del bucle.
  • - Puede contener llamadas a funciones o métodos, que se ejecutarán en cada iteración.

Ciclos While

Dentro de la programación, el ciclo while es una herramienta esencial que nos permite repetir una serie de instrucciones mientras una condición sea verdadera. Este tipo de bucle es especialmente útil cuando no se conoce con anticipación cuántas veces se ejecutará el ciclo.

El ciclo while evalúa una condición booleana antes de ejecutar el cuerpo del ciclo. Si la condición es true, se ejecuta el código dentro del bucle; si es false, se detiene la ejecución.

A diferencia del bucle for, el while no tiene una estructura predefinida para el inicio, condición y modificación de la variable. Todo esto debe controlarse manualmente dentro del bucle.

Este tipo de bucle es útil cuando, por ejemplo, estamos esperando una entrada del usuario o una condición que no sabemos cuándo se cumplirá.

Sintaxis básica:


    while (condición) {
        // Instrucciones a ejecutar mientras la condición sea true
        // Debes asegurarte que la condición cambie en algún momento
    }

En los bucles while es importante controlar la condición dentro del cuerpo, es muy normal que estos bucles se ejecuten una cantidad infinita de veces trayendo problemas al programa. Para ello podemos determinar una serie de pasos.

  • 1. Definir una variable fuera del bucle while.
  • 2. Establecer una condición en el bucle usando la variable anteriormente definida.
  • 3. Modificar de alguna forma la variable dentro del bucle while, con la posibilidad de dejar de cumplir la condición anterior y asi salir del bucle.

Tambien puedes utilizar una condición siempre verdadera y utilizar break dentro del codigo para dejar de ejecutarlo.

Algunas características del cuerpo del bucle while incluyen:

  • - El uso de variables de control externas, que deben ser actualizadas manualmente para evitar ciclos infinitos.
  • - Se puede incluir una instrucción if para verificar condiciones dentro del bucle.
  • - Se puede utilizar break para salir del bucle antes de que la condición sea falsa.
  • - El uso de continue permite saltar la ejecución del resto del cuerpo en una iteración y continuar con la siguiente.
  • - Se puede usar en combinación con estructuras de datos, lectura de archivos o entrada del usuario.

Un ejemplo práctico sería contar desde un número inicial hasta un valor final, controlando la condición manualmente: 


    int i = 0;  // variable de inicio (bandera)

    while (i < 5) {
        System.out.println(i);
        i++; // modificador para evitar bucle infinito
    }

En este ejemplo, la variable i se incrementa en cada iteración y el ciclo se detiene cuando i ya no cumple la condición i < 5.
