Funciones

En Java, una función (también llamada método) es un bloque de código reutilizable que realiza una tarea específica. Su sintaxis básica es: 


    public static tipoDeRetorno nombreFuncion(parámetros){
        // Cuerpo de la función
        return valor;
    }

Explicación

  • - tipoDeRetorno: El tipo de dato que devuelve la función (como int, String, etc.). Si no devuelve nada, se usa void.
  • - nombreFuncion: El nombre que identifica a la función.
  • - parámetros: Opcionales, son variables que la función puede recibir para trabajar con ellas.

Ejemplo básico:


    public static int sumar(int a, int b){
        // Cuerpo de la función
        return a + b;
    }

Ejercicios: Funciones

En esta sección encontrarás ejercicios prácticos para que domines las funciones en Java.

Nivel Fácil

Ejercicio: Suma de dos números

Crea una función que reciba dos números enteros y devuelva su suma.

Ejercicio: Número par o impar

Crea una función que reciba un número entero y devuelva true si es par o false si es impar.

Nivel Intermedio

Ejercicio: Contar vocales

Crea una función que reciba una cadena de texto y cuente cuántas vocales contiene.

Ejercicio: Número primo

Crea una función que determine si un número entero es primo.

Nivel Avanzado

Ejercicio: Invertir una cadena sin usar funciones de librería

Escribe una función que reciba una cadena y devuelva la misma cadena invertida sin usar funciones como reverse().

Ejercicio: Fibonacci Recursivo

Crea una función recursiva que devuelva el n-ésimo número de Fibonacci.
